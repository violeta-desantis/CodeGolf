# Code Golf: Triangle
A C program that prints an ASCII triangle of the given height, written using as few characters as possible.

compile using:
> gcc main.c -o out

example usage:
> ./out 1
```
/\
```

> ./out 2
```
 /\
/__\
```

> ./out 3
```
  /\
 /  \
/____\
```

> ./out 4
```
   /\
  /  \
 /    \
/______\
```
